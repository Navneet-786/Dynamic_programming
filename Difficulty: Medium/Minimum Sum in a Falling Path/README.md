<h2><a href="https://www.geeksforgeeks.org/problems/minimum-sum-in-a-falling-path/1">Minimum Sum in a Falling Path</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">Given a matrix <strong>mat[][] </strong>of size n*n,&nbsp;<span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">where each cell contains an integer value. A </span><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">falling path</span><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;"> starts from any cell in the first row and moves downward to the last row.</span></span></p>
<p><span style="font-size: 14pt;">&nbsp;From a cell <code data-start="353" data-end="361">(i, j)</code>, you may move to the cell directly below <code data-start="403" data-end="413">(i+1, j)</code>, the cell diagonally down-left <code data-start="445" data-end="457">(i+1, j-1)</code>, or the cell diagonally down-right <code data-start="506" data-end="518">(i+1, j+1)</code> if they exist. Your task is to compute the <strong data-start="561" data-end="586">minimum sum of values</strong> along any valid falling path from the top row to the bottom row.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>mat[][] = [[5, 10],<br>               [25, 15]]
<strong>Output: </strong>20
<strong>Explanation: </strong>The minimum falling path is <code data-start="800" data-end="808">5 → 15</code> with sum <code data-start="818" data-end="822">20</code>.<br></span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>mat[][] = [[1, 2, 3],<br>                [4, 5, 6]<br>                [7, 8, 9]]
<strong>Output: </strong>12
<strong>Explanation: </strong>The minimum falling path is <code data-start="983" data-end="994">1 → 4 → 7</code><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">, with sum 12</span><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">.</span> <br></span></pre>
<p><span style="font-size: 14pt;"><strong>Constraints:</strong></span><br><span style="font-size: 14pt;">1 ≤ n ≤ 100<br>-100 ≤ mat[i][j] ≤ 100</span></p></div>
